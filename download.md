# 0-DAY ACADEMY — Course Modules

## Module 01: Network Security
TCP/IP deep dives, scanning, sniffing, firewall evasion, and man-in-the-middle attacks.

## Module 02: Web App Hacking
SQL injection, XSS, CSRF, SSRF, authentication bypasses, and API exploitation.

## Module 03: Privilege Escalation
Linux & Windows privesc vectors, kernel exploits, misconfigurations, and SUID abuse.

## Module 04: OSINT
Open-source intelligence gathering, social media recon, metadata analysis, and dorking.

## Module 05: Cryptography
Symmetric & asymmetric ciphers, hashing, PKI, and real-world crypto attack patterns.

## Module 06: Reverse Engineering
Disassembly, debugging, binary patching, malware analysis, and anti-reversing techniques.

## Module 07: Social Engineering
Phishing campaigns, pretexting, vishing, physical security, and human-factor exploits.

## Module 08: CTF Challenges
Hands-on capture-the-flag labs across web, pwn, crypto, forensics, and misc categories.
